# osl-frontend
The frontend for OSL. Live example: https://osl.halcyon.moe<br />
The code isn't perfect, but it works.

## Running the development server:
```sh
npm run dev
```
or
```sh
yarn run dev
```

## Running production:
```sh
npm run build
npm run start
```
or
```sh
yarn run build
yarn run start
```
